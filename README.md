# File Scanner for Windows 11

Мощный инструмент для сканирования файловой системы Windows 11 с визуальным прогресс-баром. 
Позволяет находить все файлы с указанным расширением по всей системе.

## ✨ Возможности

- 🔍 **Полное сканирование системы** - поиск по всем доступным дискам (C:, D:, E: и т.д.)
- 📊 **Визуальный прогресс-бар** - наглядное отображение процесса сканирования
- 🚀 **Многопоточность** - ускоренное сканирование с использованием до 16 потоков
- 🎨 **Цветной вывод** - улучшенная читаемость результатов в консоли
- 💾 **Сохранение результатов** - автоматическое сохранение списка найденных файлов
- 🛡️ **Обработка ошибок** - корректная обработка недоступных папок и файлов
- ⚡ **Прерывание по Ctrl+C** - безопасное завершение сканирования
- 📝 **Детальная статистика** - время выполнения, количество найденных файлов

## 📦 Требования

- **Операционная система**: Windows 11 (также работает на Windows 10/8/7)
- **Python**: версия 3.6 или выше
- **Права администратора**: рекомендуется для полного доступа ко всем папкам

## 🚀 Использование
**Быстрый старт**
**1. Запустите скрипт** от имени администратора (рекомендуется):
Запустите cmd, перейдите в каталог где находится скрипт.
Напишите python find_scanner.py

**2. Введите расширение файла для поиска (без точки):**

**3.Дождитесь завершения сканирования. Процесс включает:**

Автоматическое определение всех дисков
Сбор директорий на каждом диске
Многопоточное сканирование с прогресс-баром
Отображение найденных файлов в реальном времени

**4. Просмотрите результаты:**

Все найденные файлы отображаются в консоли
При желании сохраните результаты в текстовый файл

## ⚙️ Параметры настройки
Вы можете изменить параметры сканирования в коде:

**Количество потоков (по умолчанию 16)**
scanner = FileScanner(extension, max_workers=16)

**Ширина прогресс-бара**
progress = ProgressBar(total_dirs, width=50)

**Исключаемые папки (добавьте в метод get_directories)**
dirs[:] = [d for d in dirs if not d.startswith('$') and not d.startswith('Windows')]
